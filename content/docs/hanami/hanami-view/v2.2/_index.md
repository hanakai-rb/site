# Hanami::View

A view layer for [Hanami](//doc/hanami)

## Rubies

**Hanami::View** supports Ruby (MRI) 3.1+

## Installation

Add this line to your application's Gemfile:

```ruby
gem "hanami-view"
```

And then execute:

```shell
$ bundle
```

Or install it yourself as:

```shell
$ gem install hanami-view
```
