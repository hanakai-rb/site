## Rubies

__Hanami::View__ supports Ruby (MRI) 3.1+

## Installation

Add this line to your application's Gemfile:

```ruby
gem "hanami-view"
```

And then execute:

```shell
$ bundle
```

Or install it yourself as:

```shell
$ gem install hanami-view
```
