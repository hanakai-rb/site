# Hanami::DB

The database layer for [full-stack Hanami 2.2 applications](//doc/hanami).
It's a thin layer on top of [ROM](//doc/rom) 5.

## Installation

**Hanami::DB** supports Ruby (MRI) 3.1+

Add this line to your Hanami application's Gemfile:

```ruby
gem "hanami-db"
```

And then execute:

```shell
$ bundle
```
